# Treatment Protocols

## Mycotic Nails

First Visit:
PAS stain.
Debride nails.
Second Visit:
2 weeks Review PAS results.
If positive option of Lamisil (Run LFTs).
Dispense topical antifungal.
Third Visit
Return to office, as necessary.

At Risk Foot Care (Nails and Calluses)
Debride Mycotic Nails 11720 or 11721.
Do PAS stain and will treat Mycotic Nails.
If 11720, also bill 11719 for elongated nails with at risk code
Include Q modifier with all codes Hyperkeratosis 11055, 11056, 11057.
*When billing these codes, use -59 Modifier on nail codes.
For Diabetic patients, evaluate for shoes and insoles for those who qualify.
Review with patient and dispense at risk lower extremity education, oral and written.

## Heel Pain

First Visit:
X-Ray if forefoot pain also, take 2 views of the foot (73630) and two views X-Rays of heel (73650).
Night Splint/ DME forms. (If not dispensing night splint this visit can dispense Airheel.)
Airheel or Strapping.
Corticosteroid Injection.
Dispense topical analgesics and patient education sheet.
Discuss custom made orthotic devices.
Prescription for NSAIDs, if warranted.
Second Visit:
One to two weeks out.
Cast for custom made orthotic devices.
Dispense Airheel/Strapping and or Night splint if not dispensed 1st visit and has AM pain.
Third Visit:
Two-three weeks later.
Dispense custom made orthotic devices.
Second Corticosteroid injection if needed.
If pain level has increased and difficulty ambulating consider pneumatic walker and MRI.
Notes:
Consider a third corticosteroid injection if needed.
Always dispense pneumatic walker before considering surgery.
At first visit, if pain and symptoms are extreme, consider MRI. No injection first visit and dispense pneumatic walker.
Refer to Physical Therapy if pain is not resolved.

## Neuroma

### First Visit:

3 View X-Ray of the foot.
Corticosteroid injection.
Strapping.
Patient education sheet.
Discuss custom made orthotic devices.
Dispense topical analgesic.

### Second Visit:
Cast for custom made orthotic devices.
Strapping.
Third Visit:
Dispense custom made orthotic devices.
Corticosteroid injection.
If not responsive to treatment, consider neurolysis and if non-responsive, consider surgery.

## Achilles Tendonitis:

### First Visit:
X-Ray.
Pneumatic walker.
Dispense topical analgesic.
Prescription for NSAIDs.
*If mild, dispense ankle brace and bilateral heel lifts. Discuss custom made orthotic devices. Order MRI if indicated.

### Second Visit:

Cast for custom made orthotic devices.
Dispense night splint.
Third Visit:
If discontinuing Pneumatic walker, dispense ankle brace.
If pain continues, refer to Physical therapy.

## Foot/ Ankle Sprain

### First Visit

X-Ray
If mild, dispense ankle brace and if moderate to severe dispense pneumatic walker.
Prescription for NSAIDs.
Dispense topical analgesic.
Second Visit/ Third Visit:
If in pneumatic and symptoms decreased depending on degree, dispense ankle brace and if indicated, refer to Physical Therapy.
Dispense over the counter arch supports (Foot Steps, etc.).

## Foot/ Ankle Fracture:

First Visit: 
X-Ray. 
Pneumatic walker (if forefoot fracture with edema in forefoot, dispense non-pneumatic walker).
Dispense topical analgesic.
Prescription for NSAIDs.
Second Visit:
X-Ray.
Continue with walker.
Third Visit:
X-Ray.
If fracture healed:
If it’s an ankle fracture, dispense ankle brace and arch supports (Foot Steps).
If foot fracture, dispense arch support (Foot Steps).
Refer for therapy if indicated.
If fracture is not healed, continue with walker.

Fourth Visit:
See treatment in visit three.
If symptoms continue, order further radiology testing.




Notes:
Written condition education sheets are attached. Please have ready for the patients.
Dispense Foot Steps when warranted, but never substitute for custom made orthotic devices unless patient refuses custom.
Send for Physical Therapy for when conditions are not responding to treatment.
Consider further diagnostic testing if conditions are not improving (MRI, CAT scan, bone scan, etc.)
Examine all existing and new patients for suspicious looking skin lesions and biopsy. Always best to be safe.















Infected Ingrown Nail
First Visit:
Partial nail avulsion. 
Note: Proper coding is 11730 and not 10060
Second Visit:
If ingrown is recurrent, perform Matrixectomy if infection has been resolved. If Matrixectomy was performed, see patient back in one week for follow up and two weeks after that.
Note:
When a Matrixectomy is performed, consider Amerigel Post-Op surgical kit.

Bunion/ Hallux Limitus
Visit one:
X-Ray
Strapping 
If painful, corticosteroid injection.
Dispense Dr. Jill’s Bunion pad.
Discuss custom orthotics
Visit two:
Casting for custom made orthotic devices.
Strapping.
If chronic problem, perform surgical consultation.
Note:
If Bunionectomy is performed, dispense non-pneumatic walker.
