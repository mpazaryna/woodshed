# ADK Starter - 002

https://www.youtube.com/watch?v=RFFcBkSupxk

```bash

# Create
python -m venv .venv

# Activate (each new terminal)
source .venv/bin/activate

# install requirements
pip install -r requirements.txt

# bootstrap app
adk init app

```


## Run agent

```bash
adk run app
```

## Run web server

```bash
adk web
```
